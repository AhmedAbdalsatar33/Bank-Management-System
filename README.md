# 🏦 Bank Management System (C++)

A console-based **Bank Management System** developed using **C++ and Object-Oriented Programming (OOP)**.  
This project simulates real-world banking operations such as client management and financial transactions using file-based data storage.

---

## 🎯 Project Purpose
The system was built to simulate core banking operations in a simplified but structured way, focusing on applying OOP principles, file handling, and modular design in C++.

---

## 🚀 Key Features

- Secure user authentication system  
- Role-based access control (permissions system)  
- Client management (add, update, delete, search)  
- Banking transactions (deposit, withdraw, transfer)  
- Currency exchange module  
- Activity logging system  
- Persistent file storage using `.txt` files  

---

## 🛠️ Technologies Used

- C++
- Object-Oriented Programming (OOP)
- File Handling
- Modular Architecture  

---

## 🧠 System Highlights

- Clean OOP structure using classes and encapsulation  
- Separation of logic into modules  
- File-based persistence without database dependency  
- Console-based interactive UI  

---

## 📌 Future Improvements

- Add database support instead of file storage

- Multi-user concurrent system
